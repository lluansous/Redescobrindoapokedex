Redescobrindo a Pokedex

A Pokédex é um item indispensável na jornada de qualquer treinador. Nela estão contidas as informações de todos os pokémons, porém, apenas um por vez. Mas o que os dados presentes nela poderiam nos contar, se pudéssemos ter uma visão geral dela?

Bem, logo de cara precisamos lhe contar que os monstrinhos de bolso parecem não seguir as regras da física do mundo real. Isso fica claro quando observamos que não há correlação entre peso e velocidade, ou entre ...

Credit: Projeto de Visualização de Dados by Luan Sousa

Darwin provavelmente estaria intrigado ao perceber que a seleção natural do mundo Pokémon parece favorecer alguns tipos em detrimento de outros... ou, talvez seja apenas uma preferência muito especifica dos game designers...

Credit: Projeto de Visualização de Dados by Luan Sousa

Os Pokémon voadores, por exemplo, parecem quase não existir como tipo principal, mas são os que mais se destacam em termos de tipo secundário, principalmente quando analisamos o número de conexões existentes entre os tipos.

Credit: Projeto de Visualização de Dados by Luan Sousa

E se combinarmos as visualizações, algumas coisas ficam bem mais interessantes! Os tipos Psychic e Steel parecem ter mais outliers em termos de peso e velocidade do que os outros.

Credit: Projeto de Visualização de Dados by Luan Sousa

Em algumas gerações, aparentemente, os desenvolvedores tiveram surtos criativos e criaram muitos novos Pokémons, como por exemplo a 5ª e a 1ª, em outras, se contentaram apenas em ficar na média (2ª e 6ª).

Credit: Projeto de Visualização de Dados by Luan Sousa

Chega a ser interessante como eles parecem seguir um padrão, alternando entre gerações... Podemos até estimar que provavelmente a próxima geração (9ª) terá menos pokémons que a ultima (8ª).

Credit: Projeto de Visualização de Dados by Luan Sousa

Quando olhamos para os status totais, que representa uma forma de categorizar os Pokémons mais fortes, percebemos que em média as gerações apresentam-se bastante semelhantes com valores em torno de 430.

Credit: Projeto de Visualização de Dados by Luan Sousa

Se mudamos a perspectiva para tipo principal dos Pokémons, observamos que em média os mais fortes são os do tipo Dragon, Psychic e Steel, por sua vez, os mais fracos são os do tipo Grass, Normal e Bug.

Credit: Projeto de Visualização de Dados by Luan Sousa

Como esperado, quando nos focamos em raridade, percebemos que os lendários e Míticos são mais fortes do que os Pokémons comuns.

Credit: Projeto de Visualização de Dados by Luan Sousa
